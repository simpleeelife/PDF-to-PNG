# PDF to PNG Converter

PDFファイルをPNG画像に変換するWindows向けツール。ドラッグ＆ドロップで簡単に変換できます。

## ダウンロード

[Releases](https://github.com/simpleeelife/PDF-to-PNG/releases) から最新の `PDF_to_PNG.exe` をダウンロード

## SHA-256 検証

```powershell
certutil -hashfile PDF_to_PNG.exe SHA256
```
出力されたハッシュ値がリリースの `SHA256SUMS.txt` と一致することを確認してください。
